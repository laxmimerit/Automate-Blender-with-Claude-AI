# Cyberpunk Robot — Blender Creation Prompts

A complete log of prompts used to build a futuristic cyberpunk robot character in Blender using MCP.

---

1. Delete all mesh objects from the scene. Keep only the camera and lights. Do not create any environment.

2. Create only the torso of a stylized futuristic robot character using clean hard-surface geometry with simple humanoid proportions. Do not create arms, legs, head, or environment.

3. Create only the robot head attached to the torso with glowing blue eyes, smooth metallic surfaces, and a clean cyberpunk design.

4. Create only the left robotic arm with mechanical joints and futuristic metallic design attached naturally to the torso.

5. Create only the right robotic arm matching the style and proportions of the left arm.

6. Create only the left robotic leg with clean mechanical structure and balanced humanoid proportions.

7. Create only the right robotic leg matching the left leg design and proportions.

8. Add detailed robotic hands with articulated fingers and futuristic mechanical styling.

9. Add sleek cyberpunk shoulder armor panels with glowing blue accent lines.

10. Add detailed chest paneling, glowing energy core, and futuristic mechanical surface details to the torso.

11. Add mechanical back components and subtle futuristic energy elements to the robot character.

12. Apply realistic metallic materials with matte black surfaces, brushed steel details, and glowing blue emissive accents.

13. Refine the entire robot character by fixing proportions, smoothing geometry, aligning joints, and improving overall symmetry.

14. Pose the robot character in a powerful cinematic standing pose with slightly bent arms and confident posture.

15. Create a dark reflective floor beneath the robot character. Do not create walls or environment.

16. Add dramatic cinematic lighting with blue rim lights, soft shadows, subtle fog, and cyberpunk atmosphere.

17. Position the camera at a dramatic low angle focused on the robot character with shallow depth of field.

18. Improve overall realism with better reflections, ambient occlusion, subtle surface imperfections, and cinematic render quality.

19. Create a smooth 360 degree rotating showcase animation of the robot character using exactly 72 frames and export as MP4.
